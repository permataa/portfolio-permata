Sentiment Analysis with Review & Rating

This web-based NLP project analyzes user sentiments based on reviews and star ratings extracted from the WhatsApp app on Google Play Store. Built with the Logistic Regression model combined with TF-IDF vectorization and numerical rating input, the system classifies sentiments into positive, neutral, or negative classes. An 80/20 training-testing split was used to evaluate performance, and key metrics such as precision, recall, and F1-score are displayed for transparency. The app features an interactive interface built using Streamlit, enabling users to input text and rating for real-time sentiment predictions, along with visualizations for deeper insights.
