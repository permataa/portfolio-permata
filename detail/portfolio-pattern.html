MATLAB-based desktop application

MATLAB-based desktop application to recognize needle types using image processing and pattern classification techniques. The system is able to distinguish between sewing needles, needles, and needles based on visual features such as Hue, Saturation, Value (HSV), and object area. Users can upload needle images, perform segmentation, feature extraction, and run classification automatically through an interactive GUI interface. This application is designed to demonstrate the application of image processing and simple machine learning in the context of real object recognition.
